# StockConnect 📈

StockConnect is a premium, real-time stock trading simulation platform designed for educational and professional market training. It provides a comprehensive ecosystem for traders, market administrators, and system-level managers to interact in a secure, simulated financial environment.

## 🚀 Key Features

### 🤵 Super Admin Portal
*   **System-wide Configuration**: Manage global platform parameters and tenant settings.
*   **Audit Logging**: Comprehensive, immutable record of every administrative action for full auditability.
*   **User & Role Management**: Direct control over permissions and access levels for all platform participants.

### 🏢 Market Administrator Dashboard
*   **Session Control**: Plan and execute market opening/closing times and manage automated session behaviors.
*   **Scenario Injection**: Broadcast market-moving news and events to traders in real-time to simulate volatility.
- **Order Book & Trades**: Direct audit trail of pending orders and executed trades across all symbols.
- **Trader Leaderboard**: Comprehensive ranking of participants based on portfolio performance and net worth.

### 📊 Trader Experience
- **Real-time Discovery**: Monitoring of Rwandan and global market symbols (e.g., BK, MTN Rwanda) with live trend analysis.
- **Portfolio Management**: Holistic view of holdings, current net worth, and historical trade performance.
- **Simulated Wallet**: Secure management of RWF balances for active order execution.

## 🛠️ Technology Stack

- **Frontend**: React.js with Vite, Tailwind CSS for premium UI components, and Lucide-React for high-clarity iconography.
- **Backend**: Java / Spring Boot with Maven-based dependency management.
- **Security**: Role-Based Access Control (RBAC) and detailed audit trails.

## 📦 Getting Started

### Prerequisites
- Node.js (v18+)
- Java Development Kit (JDK 17+)
- Maven

### Frontend Installation
```bash
cd "Frontend "
npm install
npm run dev
```

### Backend Installation
```bash
cd Backend/stockconnect
mvn spring-boot:run
```
